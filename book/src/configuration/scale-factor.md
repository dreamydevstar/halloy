# Scale factor

## `[scale_factor]` Section

```toml
scale_factor = <float>
```

| Key            | Description                                             | Default |
| -------------- | ------------------------------------------------------- | ------- |
| `scale_factor` | Control the scale factor of the entire application. | `1.0`[^1]   |


[^1]: Limited between `0.1` and `3.0`.
