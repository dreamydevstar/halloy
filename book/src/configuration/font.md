# Font

## `[font]` Section

```toml
[font]
family = "<string>"
size = <integer>
```

| Key      | Description                   | Default  |
| -------- | ----------------------------- | -------- |
| `family` | Monospaced font family to use | `""`[^1] |
| `size`   | Font size.                    | `13`     |

[^1]: Iosevka Term is provided by the application, and used by default.
